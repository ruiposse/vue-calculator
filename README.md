# vue-calculator

> A simple calculator built with Vue.js

## Instructions

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server at localhost:3000
$ npm run build
$ npm run start

# run unit tests
$ npm run test
```
